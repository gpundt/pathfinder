# Pathfinder

Web scraper | Written in Rust

## How to build

To build the Pathfinder binary, use the Makefile:

1) View the available `make` targets:
    - `make help`

2) Build and relocate binary with:
    - `make`


## How to Run

To run Pathfinder, do the following:

1) Ensure pathfinder is accessible via PATH:
    - `which pathfinder`

2) View the available Pathfinder options:
    - `pathfinder -h`

3) Run Pathfinder targeting a specific URL:
    - `pathfinder -u http://google.com`
